# Manuscript tables

This folder contains the manuscript-aligned table files for the four-way acquired k-space residual-risk verification study.

## Main tables

| Manuscript table | File stem | Purpose |
|---|---|---|
| Table 1 | `main/table_1_dataset_split_and_evaluation_population` | Dataset subset, volume-level split, leakage checks, heterogeneity, and independent test population |
| Table 2 | `main/table_2_residual_risk_target_parameters` | Fixed parameters used for residual-risk target construction |
| Table 3 | `main/table_3_a4_cnn_ablation_summary` | CNN residual-risk predictor ablation used for A4 CNN model selection |
| Table 4 | `main/table_4_independent_holdout_verification_performance` | Independent verification performance against the held-out target |
| Table 5 | `main/table_5_paired_a4_minus_comparator_differences` | Paired A4-minus-comparator performance differences |
| Table 6 | `main/table_6_split_conformal_residual_risk_bounds` | Split conformal residual-risk coverage and interval width |
